This site. Built from scratch in Astro 5 + Tailwind v4 + MDX. Content is file-based across seven collections: posts, papers, notes, renders, library, frontpages, and projects. Every page renders as static HTML at build time; interactivity is opt-in via React islands.

## Stack

- **Astro 5** — static site generator, content layer, view transitions
- **Tailwind CSS v4** — utility styling, token-driven theme
- **MDX** — content with inline components
- **KaTeX** (build-time) for math, **Shiki** for code
- **Wilson** (GPU) for fractal applets
- **Three.js + react-three-fiber** for 3D renders
- Hosted on **GitHub Pages** with a custom domain

## Design directions shipped

- Default: elegant Medium-style broadsheet
- Chaos Edition: this one — a collage with ticker, horoscope, classifieds, obituaries, weather
