Julia sets are fractal structures defined by iterating $z \mapsto z^2 + c$ for a fixed complex constant $c$. Different values of $c$ produce wildly different shapes — from spirals to dendrites to filled discs.

The applet below renders a Julia set in real time on the GPU. **Drag the point** to change $c$ and watch the fractal morph. **Scroll to zoom**, **drag the background to pan**.

<WilsonGPUApplet
  client:load
  shader={`
    precision highp float;

    varying vec2 uv;

    uniform vec2 worldCenter;
    uniform vec2 worldSize;
    uniform vec2 c;

    void main(void)
    {
      vec2 z = uv * worldSize * 0.5 + worldCenter;

      vec3 color = normalize(
        vec3(
          abs(z.x + z.y) / 2.0,
          abs(z.x) / 2.0,
          abs(z.y) / 2.0
        )
        + .1 / length(z) * vec3(1.0)
      );

      float brightness = exp(-length(z));

      for (int iteration = 0; iteration < 200; iteration++)
      {
        if (length(z) >= 4.0)
        {
          break;
        }

        z = vec2(z.x * z.x - z.y * z.y, 2.0 * z.x * z.y) + c;

        brightness += exp(-length(z));
      }

      gl_FragColor = vec4(brightness / 10.0 * color, 1.0);
    }
  `}
  uniforms={{
    worldCenter: [0, 0],
    worldSize: [2, 2],
    c: [0, 1],
  }}
  worldHeight={3}
  minWorldWidth={0.00001}
  minWorldHeight={0.00001}
  minWorldX={-2.5}
  maxWorldX={2.5}
  minWorldY={-2.5}
  maxWorldY={2.5}
  draggables={{ c: [0, 1] }}
  onDrag={`
    wilson.setUniforms({ c: [x, y] });
    wilson.setUniforms({
      worldCenter: [wilson.worldCenterX, wilson.worldCenterY],
      worldSize: [wilson.worldWidth, wilson.worldHeight],
    });
    wilson.drawFrame();
  `}
/>

## How It Works

The fragment shader runs **per pixel**. For each pixel, we:

1. Map its screen coordinates to a complex number $z_0$
2. Iterate $z_{n+1} = z_n^2 + c$ up to 200 times
3. Accumulate $\sum e^{-|z_n|}$ as a smooth brightness measure
4. Color based on the escape trajectory

The constant $c$ comes from the draggable point. Wilson passes it as a GLSL uniform that updates in real time.

## The Math

A Julia set $J_c$ for a given $c \in \mathbb{C}$ is the boundary of the set of points $z_0$ for which the orbit $\{z_n\}$ remains bounded:

$$
K_c = \left\{ z_0 \in \mathbb{C} : \sup_n |z_n| < \infty \right\}
$$

The Mandelbrot set is the set of $c$ values for which $z_0 = 0$ stays bounded — in other words, it's a map of which Julia sets are connected.
