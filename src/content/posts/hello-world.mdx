This is the first post on my blog. I built this entire thing from scratch using Astro 5, Tailwind CSS v4, MDX, and TypeScript.

## Why Astro?

Astro ships **zero JavaScript** by default. Pages are rendered as static HTML at build time. When I need interactivity — like a 3D viewer — I use React as an "island" that hydrates only where needed.

The result: fast pages, clean architecture, full control.
