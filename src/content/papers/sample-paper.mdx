## Extended Abstract

This paper presents a unified framework for analyzing SGD convergence under various learning rate schedules. We consider the standard stochastic optimization setting where the objective function $f: \mathbb{R}^d \to \mathbb{R}$ is smooth but possibly non-convex.

## Key Results

The main theorem establishes that under a decaying learning rate schedule $\eta_t = \eta_0 / \sqrt{t}$:

$$
\mathbb{E}\left[\|\nabla f(x_t)\|^2\right] \leq \mathcal{O}\left(\frac{1}{\sqrt{T}}\right)
$$

This bound is tight and matches the known lower bound for first-order methods on smooth non-convex functions.

## Implications

The result unifies several prior convergence analyses and provides practical guidance for setting learning rate schedules in deep learning training.
